# DSC-site

DSC(Developers Student Club) - JIIT128 is a Google Developers program aimed for college students to help them build their mobile, web, machine learning, cloud skills, etc


## Installation

Installing React
```
    npm install -g react-cli react
    npm install -g create-react-app
    npx create-react-app my-app
    cd my-app
    npm start

    //Is used to update the version only such as 16.0.0 can be update to latest 16.13.0 but.. 
    //15.0.0 can't be updated to 16.13.0 or to 16.0.0 with this
    npm update react

    //Use this to go from 15.0.0 to 16.13.0
    npm install --save react@latest
    npm i react
```

Making of react app

```
    npx create-react-app <appname>  //  eg:  npx create-react-app blog
    cd <appname>  // cd blog
    npm start
```



## Run Locally

Clone the project

```bash
  git clone https://github.com/dsc-jiit-128/DSC-site.git
```

Go to the project directory

```bash
  cd DSC-site
```
